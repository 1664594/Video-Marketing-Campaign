# Video Marketing Campaign

## Overview
An end-to-end SQL analytics project focused on evaluating video marketing campaign performance, customer behavior, and advertising effectiveness. The project generates actionable insights through campaign analysis, KPI tracking, customer segmentation, and business intelligence.


## Business Objectives
- Analyze campaign performance
- Measure customer engagement
- Evaluate advertising ROI
- Track marketing KPIs
- Generate actionable business insights

## Tech Stack
- SQL
- Excel
- Git & GitHub

## SQL Concepts
- Joins
- CTEs
- Window Functions
- CASE Statements
- Aggregate Functions
- Subqueries
